# GEVP Dashboard 🏕️

<Info />

## Descripción del proyecto

Se solicitó un dashboard para la gestión de socios del Club Gimnasia y Esgrima de Villa del Parque (Natación). El mismo permitirá la carga y modificación de datos de los socios, así como también la visualización y exportación de reportes. El socio puede cargar o modificar sus propios datos, sin necesidad de la intervención de un administrador. Al registrarse un socio, el sistema envia un mail al administrador de turno para su aprobación.
